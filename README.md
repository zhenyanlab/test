## 开源理念
蜻听在线客服起源于我们自用的一个网站的小模块,随着客户的需求越提越多,日积月累,就有了这个开源项目.
至于为什么开源?我自己觉得....
+ 程序员都喜欢源码,可以看,可以改,自己编出来的东西运行起来也更踏实.
+ 如果很多人都有这样的需求，而又有人能把它实现，那这件事就必然会发生
+ 有啥问题或者有什么新想法,欢迎提PR,欢迎联系我们.(联系方式放在最后)
